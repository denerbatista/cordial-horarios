Como obter um PAT e instalar o runner

1) Gerar o PAT (token de acesso pessoal):

  1. Vá em github.com/settings/personal-access-tokens e clique em Generate new token (Fine-grained).
  2. Repository access → Only select repositories → cordial-horarios.
  3. Repository permissions → Actions: Read & write.
  4. Copie o token github_pat_… e cole aqui em cima.

2) Instalar o runner (1 vez):

  1. Vá em Settings → Actions → Runners → New self-hosted runner.
  2. Escolha Linux x64 (recomendado rodar no WSL Ubuntu) e siga os comandos exibidos.
  3. Quando perguntar Enter additional labels, digite cordial.
  4. Instale como serviço: sudo ./svc.sh install && sudo ./svc.sh start (Linux/WSL) ou rode install.cmd (Windows).

Via servidor local fallback

Sem PAT: roda direto no localhost:8765 se você abriu o sync-server.

Como rodar o servidor local
  1. Na sua pasta privada ~/cordial-source/, rode python scripts/sync_local.py --server.
  2. Mantenha a janela aberta. Volta aqui — o status deve virar verde.

Sobre a consulta de horários da Cordial Turismo

Este site agrega os horários dos ônibus da Cordial Transportes e Turismo (Espírito Santo) em um único lugar pesquisável. Os PDFs oficiais são baixados todos os dias e o conteúdo é estruturado em um banco de horários consultável por origem, destino, dia da semana e linha. Você pode ver os próximos ônibus saindo agora de qualquer cidade ou bairro coberto, sem precisar abrir o PDF da empresa.

Perguntas frequentes

Como consultar o horário do ônibus da Cordial?

Escolha a localidade de origem e o destino no formulário do topo. Para ver o que sai agora, troque para a aba Próximos ônibus. O filtro de dia da semana ajusta a lista para dias úteis, sábado, domingo ou feriado.

Os horários são oficiais?

Este site é independente. Os dados são extraídos automaticamente dos PDFs publicados pela Cordial Turismo. Em caso de divergência, sempre confira a página oficial.

Com que frequência os horários são atualizados?

Todos os dias, durante a madrugada (cerca de 03h horário de Brasília). Quando a Cordial publica um PDF novo, o site captura e atualiza o banco de horários sozinho.

O site funciona no celular?

Sim, foi desenhado primeiro para celular. Você pode salvar como aplicativo na tela inicial (PWA) para abrir mais rápido, sem precisar do navegador.

O que significam os horários marcados com asterisco?

São horários que circulam apenas em dias letivos (com aulas). No site, essas viagens recebem a etiqueta só dias letivos.

Posso saber qual linha passa em determinado lugar?

Sim. Use a aba Linhas para ver todas as linhas em operação (001, 002, 003 e assim por diante) com os bairros que cada uma cobre.

Por que algumas opções de destino somem quando escolho a origem?

O formulário filtra automaticamente os destinos para mostrar apenas pares válidos. Se um ônibus não sai de X para Y nos PDFs oficiais, a opção nem aparece para evitar confusão.

Posso comparar mais de uma saída ou destino ao mesmo tempo?

Sim. Na aba Buscar, tanto o campo de saída quanto o de destino aceitam vários valores: escolha um e ele vira uma etiqueta; adicione outros e a lista mostra os horários de todas as combinações juntas. Toque no × da etiqueta para remover. Útil, por exemplo, para ver de uma vez os ônibus de Coqueiral e Santa Cruz para Aracruz.

Como salvar meus horários favoritos?

Toque na estrela de qualquer horário para salvá-lo. Eles ficam reunidos na aba Favoritos e são guardados no próprio aparelho (sem precisar de cadastro). Útil para deixar à mão o ônibus que você pega todo dia.
